# Fashion MNIST Classification Using SVM and CNN

## Overview

This project explores image classification on the Fashion MNIST dataset using two different machine learning approaches: Support Vector Machine (SVM) and Convolutional Neural Network (CNN). The goal is to compare the performance of a traditional machine learning model with a deep learning model for multi-class clothing image classification.

## Dataset

The project uses the Fashion MNIST dataset, which contains grayscale images of clothing items belonging to 10 categories, including shirts, trousers, dresses, shoes, and bags.

Dataset source:
https://www.kaggle.com/datasets/zalando-research/fashionmnist

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Scikit-learn
* TensorFlow / Keras
* Jupyter Notebook

## Project Workflow

1. Data loading and preprocessing
2. Data visualization and exploration
3. SVM model training and evaluation
4. CNN model design and training
5. Performance comparison between models
6. Results analysis

## Results

Both models were evaluated on the Fashion MNIST dataset. The CNN achieved higher classification performance by learning spatial features directly from the images, while the SVM provided a strong traditional machine learning baseline.
